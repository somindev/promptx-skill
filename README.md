# PromptX Skill

Intelligent prompt framework matcher for Claude Code. Matches user inquiries to the most appropriate prompt framework from 57 curated frameworks.

## Installation

Copy the `promptx` folder to your Claude Code skills directory:

```bash
cp -r promptx ~/.claude/skills/
```

## Usage

In Claude Code, simply ask about prompt engineering or framework selection:

- "Help me write a prompt for..."
- "What prompt framework should I use for..."
- "Guide me on structured prompting..."

## Features

- 57 curated prompt frameworks
- Intelligent matching based on user intent
- Supports various use cases: creative writing, analysis, coding, and more
